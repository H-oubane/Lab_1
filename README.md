# Lab Android – HelloToast

Application Android simple réalisée avec Android Studio en Java.

## Objectif
Créer une application qui affiche un message Toast et un compteur.

## Fonctionnalités
- Bouton **"Afficher un message"** → affiche un Toast "Bonjour !"
- Bouton **"Incrémenter le compteur"** → incrémente un compteur affiché à l'écran

## Structure du projet
```
HelloToast/
 └── app/
      └── src/
           └── main/
                ├── java/com/example/hellotoast/
                │    └── MainActivity.java   → logique Java
                └── res/
                     └── layout/
                          └── activity_main.xml  → interface XML
```

## Technologies utilisées
- Android Studio
- Java
- XML (interface)
- API minimum : 24 (Android 7.0)
